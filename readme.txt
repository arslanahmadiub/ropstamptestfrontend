# ropstamp test frontend

## Description

`ropstamp-project` is a modern, efficient, and feature-rich React.js application developed using Vite. The project is set up with a number of high-quality, mature libraries for building dynamic user interfaces, handling HTTP requests, form handling, and more.

The application is designed to be highly performant, scalable, and easy to maintain. It makes use of modern JavaScript features and follows best practices for React.js development.

## Features

- Lightning-fast development with `vite`
- Styling with `@emotion/react` and `@emotion/styled`
- Material-UI components with `@mui/material` and `@mui/icons-material`
- HTTP requests with `axios`
- Form handling with `formik` and `formik-material-ui`
- Data tables with `material-react-table`
- Routing with `react-router-dom`
- Schema-based form validation with `yup`

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

To run this project, you will need:

- Node.js installed (version 14 or later recommended)

### Installing

1. Clone the repository to your local machine:



2. Change into the project directory:

```
cd vite-project
```

3. Install the dependencies:

```
npm install
```

### Running the application

In the project directory, you can run:

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production.
- `npm run lint`: Runs the linter.
- `npm run preview`: Locally previews production build.
